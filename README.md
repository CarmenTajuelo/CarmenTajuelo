üîó **Repositories:**  
- [Frontend (React + MUI + Zustand)](https://github.com/SprintFlow/SprintFlow-Client)

### üöÄ Key Highlights

- **End-to-End Full Stack**: Frontend in React with Material-UI, Zustand for state management; Backend in Node.js/Express with MongoDB.

- **Role-Based Access & Security**: JWT authentication, Admin vs Developer permissions, password hashing with bcrypt.
